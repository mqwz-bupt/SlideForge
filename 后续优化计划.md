# SlideForge 后续优化计划

## 项目现状

SlideForge v1.0 已完成核心功能：
- 向导式 6 步流程（主题 → 范围 → 大纲 → 氛围 → 风格 → 生成）
- AI 多模型支持（DeepSeek / 通义千问 / GLM-4）
- 12 套风格预设 × 12 种布局模板
- 三格式导出（HTML / PPTX / PDF）
- AI 对话式幻灯片修改
- SQLite 本地持久化
- 中英文 i18n + 深色主题
- 全屏预览 + 缩略图 + 同步滚动

---

## P0 — 核心功能补全

### 1. PDF / DOCX 文档解析

**现状**：仅支持 TXT / MD 纯文本上传

**目标**：支持 PDF 和 DOCX 文件上传，提取文本后走 document path 生成大纲

**改动文件**：
- `src/main/index.ts` — 新增 `file:openPdf`（pdf-parse）和 `file:openDocx`（mammoth.js）IPC handler
- `src/renderer/src/features/wizard/steps/TopicStep.tsx` — 文件选择器增加 PDF/DOCX 格式

**验证**：上传 PDF → 大纲生成 → 幻灯片正常

### 2. 应用图标 + 安装包

**现状**：`package.json` 中 `"icon": null`，打包的 exe 无图标

**目标**：设计 SlideForge 图标，完成 NSIS 安装包

**改动文件**：
- 新增 `build/icon.ico`（256×256 应用图标）
- `package.json` — `build.win.icon` 指向图标文件

**验证**：`npm run dist` → 安装包有正确图标

## P1 — 体验提升

### 3. 拖拽上传文件

**现状**：TopicStep 只有按钮点击上传

**目标**：支持拖拽文件到上传区域

**改动文件**：
- `TopicStep.tsx` — 添加 drop zone，onDragOver/onDrop 事件，拖拽高亮

**验证**：拖拽 PDF 到 TopicStep → 自动上传并解析

### 4. AI 对话增强 — 增删幻灯片

**现状**：ChatPanel 只能修改内容，不能增删页或改布局

**目标**：AI 对话支持「增加一页」「删除第3页」「改为 timeline 布局」

**改动文件**：
- `projectStore.ts` — 新增 `addSlide(index)` / `removeSlide(index)` actions
- `ChatPanel.tsx` — 扩展 slide-updates 协议支持 `add` / `delete` / `changeLayout`

**验证**：对话「增加一页关于XX的幻灯片」→ 新页出现

### 5. 硬编码颜色清理

**现状**：13 个文件散落 `#6750A4` 等硬编码色值，深色模式不完整

**目标**：所有颜色统一走 `theme.colors.*`

**改动文件**：
- 全局搜索替换硬编码色值为 theme 引用
- 重点：Toolbar 图标色、SettingsDialog 图标色、DoneIcon 渐变色

**验证**：深色主题下所有 UI 元素颜色一致

### 6. 导出体验优化

**现状**：导出无进度提示，用户等待时无反馈

**目标**：导出时显示 loading，完成后弹出文件目录

**改动文件**：
- `Toolbar.tsx` — 导出按钮点击后显示 loading 状态
- 导出完成后 toast 通知 + `shell.showItemInFolder()`

**验证**：点击导出 → 看到 loading → 完成后弹出文件夹

## P2 — 锦上添花

### 7. 幻灯片拖拽排序

**现状**：编辑器中不能调整幻灯片顺序

**目标**：缩略图条支持拖拽排序

**改动**：PreviewArea 添加拖拽事件，projectStore 新增 `reorderSlides`

### 8. 用户自定义风格预设

**现状**：只有 12 套内置风格

**目标**：用户可调整配色、字体，保存为自定义预设

**改动**：新增 StyleEditor 组件，自定义预设存入 SQLite

### 9. 演讲者备注

**现状**：Slide 数据模型有 `notes` 字段但 UI 未展示

**目标**：大纲面板编辑备注，导出 PPTX 时写入 speaker notes

**改动**：OutlinePanel 添加备注区域，PPTX 导出填充 notes

### 10. 快捷键系统

**现状**：只有全屏预览有快捷键

**目标**：Ctrl+S 保存、Ctrl+E 导出、Ctrl+N 新建

**改动**：App.tsx 注册全局 keyboard listener

### 11. 自动更新

**现状**：无自动更新机制

**目标**：集成 electron-updater，发布新版后自动提示更新

**改动**：package.json 添加 publish 配置，主进程集成 autoUpdater
